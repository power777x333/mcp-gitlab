## 📋 About mcp-gitlab

mcp-gitlab is a Windows app designed to help you work with GitLab without needing technical skills. It connects with the GitLab API to give you access to 76 useful tools. These tools support daily tasks like handling projects, managing merge requests, checking pipelines, and reviewing code. The app also helps with CI/CD, approvals, and issues, making your workflow smoother.

You do not have to know programming to use mcp-gitlab. It is built for users like you who want to manage GitLab projects and tasks in one place.
